# Hack Knight Project

This repository contains both the backend and frontend parts of the Hack Knight project. The backend is a Python-based server application that handles server-side logic and API endpoints, while the frontend is a modern web application built with TypeScript and Tailwind CSS.

## Overview

The Hack Knight project is divided into two main parts:

1. **Backend:** A Python-based server application that manages core logic, API endpoints, and communication with databases and external services.
2. **Frontend:** A web application built with TypeScript and styled using Tailwind CSS, providing a responsive and interactive user interface.
