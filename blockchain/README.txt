# Frontend Setup and Running Instructions

## Prerequisites
- Node.js (version 14 or higher)
- npm (comes with Node.js) or Yarn

## Setup

### Navigate to the Frontend Directory
```bash
cd hack-knight-25/frontend
Install Dependencies
Using npm:

-bash
-Copy
-npm install
-Or using Yarn:

-bash
-Copy
-yarn install
-Running the Frontend
-Start the Development Server
-Using npm:

-bash
-Copy
-npm run dev
-Or using Yarn:

-bash
-Copy
-yarn dev
-File Structure
-tailwind.config.js – Tailwind CSS configuration.
-tsconfig.json – TypeScript configuration.
-Other directories/files for components, assets, and styles.
-Troubleshooting
-Installation Issues: Ensure you have the required Node.js version installed.
-Build Errors: Check terminal output during installation or startup for any errors.
-Configuration Issues: Verify that your tailwind.config.js is properly set up if styles are not being applied.
